# Projeto de Divulgação de Um Escritório de Advocacia

É um projeto simples que busca promover e aumentar a visibilidade de um escritório de advocacia. Este projeto inclui a criação de um site profissional e intuitivo, com informações detalhadas sobre os serviços oferecidos, a equipe de advogados, casos de sucesso e depoimentos de clientes.
